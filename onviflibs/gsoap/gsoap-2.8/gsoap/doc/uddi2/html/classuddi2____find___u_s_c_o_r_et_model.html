Detailed Description

Schema urn:uddi-org:api_v2:"find_tModel".

Represents a request to locate a list of tModel entries that match a set of specific criteria. The result of a search is a uddi2__tModelList object that contains information about registered tModel data that matches the criteria.

See http://uddi.org/pubs/ProgrammersAPI_v2.htm#_Toc25137716
